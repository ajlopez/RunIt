# RunIt

Run runit modules

## Installation

Via npm on Node:

```
npm install runit -g
```

## Usage

Run
```
runit <module>:<verb> [args...]
```

## Development

```
git clone git://github.com/ajlopez/RunIt.git
cd RunIt
npm install
npm test
```
